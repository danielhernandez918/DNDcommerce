DND commerce Project created using Java, MySQL as the database, Stripe, and own API. Styling using personal CSS and bootstrap.

Allow users to view and search for items in website. Depending if users are logged in or not, will have different experiences. Allow users to items into cart and wishlist. Allow users to checkout with carted items.
